## Decisions Made
- BytesIO buffer approach for in-memory PDF generation (no temp files, clean Cloud Run compatibility)
- Font registration at module load via _register_fonts() — avoids per-render TTF parsing; graceful fallback
- asyncio.to_thread wraps synchronous GCS client to keep async event loop unblocked
- _wrap_text uses stringWidth from reportlab.pdfbase.pdfmetrics for accurate multi-line word wrap
- dm_keyword normalized post-LLM-generation with .upper()[:8] to guard against non-compliant output
